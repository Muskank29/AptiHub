# AptiHub

AptiHub is a group project consisting of a web application and a mobile application connected to the same Firebase backend.

## Contribution
- Web Application: Developed by me using Node.js, Express, EJS, and Firebase Firestore.
- Mobile Application: Developed by my teammate and connected to the same Firebase database.

## Features
- User authentication
- Topic-wise and difficulty-based aptitude quizzes
- Admin panel for managing questions
- Account deletion request feature
- Shared Firebase backend for web and mobile app

## Tech Stack
- Node.js
- Express.js
- EJS
- Firebase Firestore
- Bootstrap

## How to Run (Web)
1. Install dependencies: `npm install`
2. Start server: `node app.js`
3. Open: `http://localhost:8080`
